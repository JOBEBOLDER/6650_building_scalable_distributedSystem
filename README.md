# Scalable Distributed Systems

This repository contains projects for CS 6650 Scalable Distributed Systems course at Northeastern University, Boston, professor: Prasad Saripalli.

## Projects

### Assignment 1: Socket Programming
Implementation of basic socket programming concepts using TCP and UDP.

### Project 1: Key-Value Store
A simple key-value store implementation with client and server components.

### Project 2: Multi-threaded Key-Value Store using RMI
Extension of Project 1 to include multi-threading and Remote Method Invocation (RMI).

### Project 3: Replicated Key-Value Store with Two-Phase Commit
Further extension implementing server replication and consistency using a two-phase commit protocol.

## Technologies
- Java
- Socket Programming
- RMI (Remote Method Invocation)
- Multi-threading
- Distributed Systems Concepts
- Two-Phase Commit Protocol

## Key Features
- Data replication across multiple servers
- Consistency maintenance through two-phase commit
- Fault tolerance mechanisms
- Scalable architecture
